Foire Aux Questions

Toutes les questions les plus courantes

Qui sommes-nous ?

Nous sommes un groupe d'étudiants en informatique ayant choisi comme projet de fin d'étude de développer une alternative à la publicité sur internet.

AntMine est-il gratuit ?

Oui et encore mieux que ça : on vous donne de l'argent ! AntMine est une nouvelle source de revenus pour les sites web. La majeure partie de l'argent récolté pour la création de crypto-monnaie vous est destinée. Le reste finance les serveurs necessaires au fonctionnement du projet.

Est-ce compliqué à installer ?

C'est très simple. Il suffit de s'inscrire sur notre plateforme et d'ajouter le code du projet au code source de votre site web.

Quand cela sera t-il disponible ?

Bientôt ! Pour le moment le projet est toujours en phase de béta privée. Si vous êtes intéressé, n'hésitez pas à nous contacter : contact@antmine.io.

La navigation des internautes va t-elle être plus lente ?

Non, nous utilisons qu'une petite partie de la puissance de calcul de l'internaute. Hormis l'absence de publicité, ce dernier ne verra pas la différence.

Comment rejoindre l'aventure ?

Pour le moment le projet est toujours en développement. Mais si vous le souhaitez, envoyez nous un mail : contact@antmine.io.
